A simple financial planning app for people who want a clear picture of their money without the complexity of traditional finance tools.

MiniFinPlan helps you plan ahead by organizing your income, expenses, and financial goals in one place.

## Features

- Track income and expenses
- Plan your monthly budget
- Set financial goals
- Simple and distraction-free interface
- Works entirely in the browser

## Getting Started

Clone the repository:

```bash
git clone https://github.com/caestomarco/minifinplan.git
cd minifinplan
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Tech Stack

- Tanstack-React
- TypeScript
- Vite
- Tailwind CSS

## Why MiniFinPlan?

Most budgeting apps try to do everything. MiniFinPlan focuses on the basics—making it easy to plan your finances, understand where your money goes, and stay on top of your goals.
